Date Time (Moment vs Luxon)

Immutability

Moment.js object are mutable 
var m1 = moment();
var m2 = m1.add(1, 'hours');
m1.valueOf() === m2.valueOf(); // true
Luxon objects are immutable 
var d1 = DateTime.now();
var d2 = {{ "d1.plus({hours: 1});" }}
d1.valueOf() === d2.valueOf(); // false

The Luxon Way

Date Time Now:

DateTime.now(): {{ currentLuxonDate }}

Format dd/mm/yyyy

DateTime.now().toFormat('dd/mm/yyyy'): {{ formattedLuxonDateTime }}

Luxon access the property directly

var now = DateTime.now();
var year = now.year
var month = now.month
var day = now.date

Comparisons

var now = DateTime.now();
var tomorrow = {{ "now.plus({days: 1});" }}
var yesterday = {{ "now.minus({days: 1});" }}
var tomorrow = {{ "now.plus({days: 1});" }}
var isBefore = now < tomorrow; //true
var isSameOrBefore = now <= tomorrow; //also true

The Moment Way

Date Time Now:

moment(): {{ currentDateMoment }}

Format dd/mm/yyyy

moment().format('DD/mm/yyyy'): {{ formattedMomentDateTime }}

Moment uses getters

var now = moment();
var year = now.year();
var month = now.month();
var day = now.date();

Comparisons

var now = moment();
var tomorrow = now.add(1, 'day');
var yesterday = now.subtract(1, 'day');
var tomorrow = now.add(1, 'day');
var isBefore = now.isBefore(tomorrow); //true
var isSameOrBefore = now.isSameOrBefore(tomorrow); //also true
